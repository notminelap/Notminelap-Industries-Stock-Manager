# Perfect Ergonomics Warehouse Manager

Welcome to the **Perfect Ergonomics Warehouse Manager**, a sleek, robust, and full-featured inventory management system designed for complete warehouse control. Featuring a beautiful, minimal aesthetic and high-performance architecture, this software enables businesses to effortlessly track inventory, log dispatches, monitor warehouse activity, and access powerful analytics—all from a refined, user-friendly interface.

## ✨ Key Features
- **Secure Authentication**: An interactive login portal that safeguards your warehouse data behind password protection with integrated hint systems, ensuring only authorized personnel have access.
- **Dynamic Dashboard**: A comprehensive hub for real-time warehouse data, displaying current stock metrics, recent transactions, and activity trends at a glance.
- **Advanced Inventory Control**: Effortlessly restock items or dispatch out inventory with a sophisticated ledger system tracking precise details, including Supplier/Customer Names, Delivery Addresses, and Invoice/Billing numbers.
- **Real-Time Transaction Activity Feed**: Monitor the flow of goods globally. Detailed "Ledger" tracking per product to see precise inbound (`IN`) and outbound (`OUT`) flow history.
- **Minimalist & Premium UI/UX**: Built with seamless micro-animations, glassmorphism UI traits, and high aesthetic quality optimized for both ease-of-use and visual impact.
- **Command Palette & Shortcuts**: Work faster with interactive keyboard shortcuts and hotkeys designed for power users traversing large product catalogs.

## 🛠️ Technology Stack
- **Frontend Core**: React 19, Vite, Tailwind CSS
- **Animations & Visuals**: Framer Motion, Lucide React, Canvas Confetti
- **Backend Architecture**: Node.js, Express.js 5
- **Database Layer**: MongoDB (Atlas/Local) + Mongoose for robust schema enforcement.

## 🚀 Deployment
This application is fully responsive and configured to be deployed as a 24/7 web service using platforms like **Render**.
